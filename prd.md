## 当前版本：v1.3

### 1. 产品概述
#### 1.1 产品背景
在日常工作或学习中，用户常常被无意识的分心所打断。本产品旨在通过定时提醒，帮助用户觉察当下的内心渴望，从而提升自我认知与专注力。

#### 1.2 产品定位
一款 Windows 桌面轻量级悬浮插件，以非侵入式的方式定时弹出简短提问，让用户快速选择当前的心理倾向（控制、认同、安全），辅助正念练习。

#### 1.3 目标用户
- 希望提升自我觉察的普通电脑用户
- 有正念、冥想习惯的人群
- 需要周期性打断以检查自身状态的职场人士

### 2. 功能需求
#### 2.1 核心功能
- **定时弹出提问**：按照用户设定的时间间隔（默认 5 分钟，范围 1-60 分钟整数，可自定义），从悬浮球中展开一个提问卡片。
- **三选一回答**：弹出卡片显示问题"这一秒是什么想要"，提供三个选项：控制、认同、安全。用户点击任一选项后，该选项高亮并显示选中标识（向右箭头），约 0.6 秒后自动收起，计时器重置（**仅限定时器触发的弹窗**）。
- **手动触发**：点击悬浮球可随时打开提问卡片；若卡片已打开，再次点击悬浮球可关闭卡片。
- **手动触发不影响定时器节奏**：手动点击悬浮球打开的弹窗，在用户选择选项或主动关闭后，**不重置既定的自动弹出定时器**。只有定时器触发的自动弹窗在用户完成选择后才会重置倒计时，保持既定节奏。

#### 2.2 悬浮球
- **外观**：玻璃态渐变蓝色圆形，带多层同心椭圆模拟光照效果，中央显示双白色胶囊眼睛，参考设计图风格。
- **呼吸感动画**：悬浮球在 idle 状态持续以约 1 秒为周期做轻微缩放（幅度约 4%），让产品保持"活着"的状态。该动画不引导用户呼吸，仅作为视觉呼吸感存在，恒定运行。拖动时动画自动暂停，松开后恢复。
- **交互**：
  - 鼠标左键单击（全程移动距离 < 3px）：打开/关闭提问卡片。
  - 鼠标左键拖动：移动悬浮球位置，卡片跟随移动，拖动过程丝滑无残影。
  - 鼠标右键：弹出设置菜单（设置频率、开机自启、反馈建议、退出）。
- **置顶**：始终悬浮在桌面最上层。
- **单实例**：通过绑定 127.0.0.1:38472 端口确保同时只有一个实例运行，防止多开导致双重定时器、双重声音、双重弹窗。

#### 2.3 设置菜单
- **设置频率**：打开设置窗口，输入新的弹出间隔（1-60 分钟整数），保存后即时生效，重新开始倒计时。
- **开机自启**：勾选后，程序将注册到 Windows 注册表，随系统启动自动运行；取消勾选则移除。
- **反馈建议**：打开反馈窗口，用户填写意见（必填）和邮箱（选填），提交后通过飞书机器人 Webhook 发送到指定飞书群。
- **退出**：关闭程序，保存当前配置。

#### 2.4 自动弹出音效
每次定时时间到达、自动弹出提问时，播放系统提示音（如 Windows 信息提示音），手动点击弹出时不播放。

#### 2.5 配置持久化
程序关闭或修改设置时，自动保存配置到本地 `config.json` 文件（包含：间隔分钟数、开机自启状态）。下次启动时自动加载上次配置，并对 `interval_minutes` 做防御性范围校验（限制在 1-60 之间）。

### 3. 非功能需求
- **平台**：Windows 系统（需 Python 环境或打包为 exe）。
- **性能**：占用极低，空闲时几乎不消耗 CPU。
- **交互反馈**：选择选项后必须在 0.6 秒内给予视觉确认（高亮+箭头）再消失，避免用户疑惑。
- **UI/UX**：玻璃态渐变风格，简洁、圆角、柔和阴影、蓝色系，无边框窗口，圆角矩形卡片，弹窗四角真实圆角。
- **可靠性**：单实例运行保护，防止用户意外多开。

### 4. 技术约束
- 使用 Python 内置的 `tkinter` 库，无需额外安装第三方 GUI 框架。
- 开机自启通过 `winreg` 操作注册表。
- 音效使用 `winsound` 模块。
- 反馈发送使用 `urllib` 标准库 POST 到飞书机器人 Webhook。
- 窗口透明效果基于 `-transparentcolor` 属性。
- 打包为 exe 时使用 PyInstaller 等工具。

### 5. 未来迭代考虑（可选）
- 增加桌面快捷方式或开始菜单项。
- 支持自定义问题文本和选项内容。
- 提供悬浮球颜色主题切换。
- 统计选项历史记录，生成简单图表。
- 支持多屏幕优化。
