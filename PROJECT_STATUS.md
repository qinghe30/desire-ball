# 项目状态报告 - DesireBall

- **项目名称**：DesireBall（桌面专注提问插件）
- **当前版本**：v1.3

## 已完成功能
- 圆形立体悬浮球，带阴影与高光，文字"desire"（v1.0）
- 点击悬浮球弹出提问卡片，再次点击关闭卡片（v1.0）
- 提问卡片包含三个选项：控制、认同、安全（v1.0）
- 选择选项后高亮显示，左侧出现向右箭头，0.6 秒后自动关闭并重置定时器（v1.0）
- 定时自动弹出（默认 5 分钟，范围 1-60 分钟），弹出时播放系统提示音（v1.0）
- 通过右键菜单设置弹出间隔（单位分钟，范围 1-60 整数）（v1.0）
- 开机自启开关（修改注册表实现）（v1.0）
- 配置持久化到 config.json（v1.0）
- 悬浮球可拖动，拖动时弹出的卡片自动跟随（v1.0）
- 所有弹窗（提问卡片、设置窗口）根据悬浮球位置智能定位，避免超出屏幕（v1.0）
- UI 采用苹果风格设计（圆角、浅灰背景、蓝色主色调）（v1.0）
- 悬浮球新增 1 秒周期呼吸感动画，柔和缩放保持视觉呼吸感，拖动时暂停（v1.1）
- 手动触发弹窗不再重置自动定时器，保持既定弹出节奏（v1.1）
- 悬浮球样式升级为玻璃态渐变蓝球 + 双白色胶囊眼睛，匹配设计图风格（v1.2）
- 拖动残影修复（withdraw/deiconify 方案）（v1.2）
- 按住所见弹窗修复（点击判断推迟到松开时刻）（v1.2）
- 设置频率限制 1-60 分钟整数，含加载时防御性校验（v1.2）
- 飞书反馈功能：右键菜单"反馈建议" + 反馈窗口 + Webhook 卡片消息（v1.3）

## 待优化 / 已知问题
- [ ] 打包为独立 exe 文件时需额外测试开机自启是否正常工作。
- [ ] 无系统托盘图标，退出只能通过右键菜单，用户可能找不到退出方式（已知，有意设计为简洁）。
- [ ] 在某些多屏幕或高 DPI 设置下，窗口边缘可能渲染略有瑕疵。
- [ ] 透明色 `#010101` 可能与某些桌面主题背景冲突，若用户使用纯黑背景可能看到矩形边框（极低概率）。
- [ ] 尚未添加多语言支持，目前仅中文。

## 技术债务
- 圆角矩形通过 `create_polygon` 模拟，不是真正的抗锯齿圆角，在放大时可能不够平滑。
- 拖动检测使用像素距离判断，阈值 3 像素，在触控板或高精度鼠标下偶尔会误判。
- 定时器逻辑依赖 Tkinter 的事件循环，若程序被阻塞（如长时间计算）会导致弹出延迟。
- magic number 散落各处（弹窗尺寸、呼吸参数等），未来迭代建议提取常量类统一管理。
