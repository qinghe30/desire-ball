# DesireBall

桌面专注助手。一枚悬浮在桌面右上角的蓝色小球，定时弹出简短提问，帮助你觉察当下的内心渴望，提升专注力与自我认知。

## 功能特性

- **定时弹窗提醒**：按你设定的间隔（1-60 分钟），自动弹出提问卡片
- **三选一正念练习**：控制 / 认同 / 安全，快速选择当下心理倾向
- **手动随时调用**：点击悬浮球随时打开提问，不打扰既定节奏
- **呼吸感动画**：悬浮球以 1 秒周期柔和缩放，保持视觉存在感
- **飞书反馈通道**：右键菜单"反馈建议"，直接提交意见到飞书群
- **开机自启**：支持注册表开机自启
- **零依赖**：纯 Python 标准库打造，无需安装任何第三方包

## 界面预览

悬浮球采用玻璃态渐变设计，带双白色胶囊眼睛。提问卡片为圆角白色卡片，简洁无边框。

## 快速开始

### 方式一：直接运行（需要 Python 环境）

```bash
python desire_ball.pyw
```

### 方式二：打包为 exe 后运行（无需 Python）

1. 双击运行 `build.bat`，等待打包完成
2. 在 `dist\DesireBall.exe` 找到打包好的 exe 文件
3. 双击运行即可

## 使用说明

| 操作 | 方式 |
|------|------|
| 打开/关闭提问卡片 | 点击悬浮球 |
| 移动悬浮球 | 拖动悬浮球 |
| 设置弹出间隔 | 右键菜单 → 设置频率 |
| 开关开机自启 | 右键菜单 → 开机自启 |
| 提交反馈建议 | 右键菜单 → 反馈建议 |
| 退出程序 | 右键菜单 → 退出 |
