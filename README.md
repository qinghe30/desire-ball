# DesireBall

桌面专注助手。一枚悬浮在桌面右上角的玻璃态蓝色小球，定时弹出简短提问，帮助你觉察当下的内心渴望，提升专注力与自我认知。

## 理论基础

三选项（控制 / 认同 / 安全）的设计来源于**圣多纳释放法（The Sedona Method）**。

圣多纳释放法由 Lester Levenson 在 1950-60 年代创立。该方法认为：人类所有情绪均源于三种基本欲望——**控制**（想要掌控）、**认同**（想要被认可）、**安全**（想要感到稳定）。

DesireBall 将这套经典框架简化为一枚桌面插件：在日常工作中，通过极低摩擦的三选一操作，帮助你快速识别当下被哪种欲望驱动，从而触发觉察。

## 功能特性

- **定时弹窗提醒**：按你设定的间隔（1-60 分钟），自动弹出提问卡片
- **圣多纳释放法练习**：控制 / 认同 / 安全，快速选择当下心理倾向
- **手动随时调用**：点击悬浮球随时打开提问，不打扰既定节奏
- **呼吸感动画**：悬浮球以 1 秒周期柔和缩放，保持视觉存在感
- **飞书反馈通道**：右键菜单"反馈建议"，直接提交意见到飞书群
- **开机自启**：支持注册表开机自启
- **单实例保护**：防止多开导致重复定时器
- **零依赖**：纯 Python 标准库打造，无需安装任何第三方包

## 界面预览

悬浮球采用玻璃态渐变设计，带双白色胶囊眼睛，整体有约 1 秒周期的呼吸感缩放动画。提问卡片为白色圆角卡片，简洁无边框。

## 快速开始

### 方式一：直接运行（需要 Python 环境）

```bash
python desire_ball.pyw
```

### 方式二：下载 exe 运行（无需 Python）

前往 [Releases](https://github.com/qinghe30/desire-ball/releases) 页面下载最新版本的 `DesireBall.exe`，双击直接运行。

### 方式三：自行打包

1. 安装 PyInstaller：`pip install pyinstaller`
2. 双击运行 `build.bat`
3. 在 `dist\DesireBall.exe` 找到打包好的 exe 文件

## 使用说明

| 操作 | 方式 |
|------|------|
| 打开/关闭提问卡片 | 点击悬浮球（全程移动距离 < 3px） |
| 移动悬浮球 | 拖动悬浮球 |
| 设置弹出间隔 | 右键菜单 → 设置频率 |
| 开关开机自启 | 右键菜单 → 开机自启 |
| 提交反馈建议 | 右键菜单 → 反馈建议 |
| 退出程序 | 右键菜单 → 退出 |
